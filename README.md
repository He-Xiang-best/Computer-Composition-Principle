## 学习建议

- 适当提前预习，可以暑假阶段先学着
- 理解背诵的点比较多，有些也会比较抽象
- 在学校上课可以当温习再巩固一遍
- 每学一个单元，结合分享给大家的习题复习巩固

## 计组实验

- 不会就是不会，除非你很有兴趣，或者是高手
- 简单应付、不用花心思、意义不大
